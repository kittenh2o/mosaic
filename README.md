## How to Use
1. Place your tile images in main/test/image_resources
2. Modify run_mosaic.py accordingly (e.g. target_uri, tile_size)
3. Run run_mosaic.py

## 如何使用
1. 将拼图图片放在 ./main/test/image_resources
2. 修改 run_mosaic.py (例如 target_uri, tile_size)
3. 运行 run_mosaic.py
